FPGA-Based 4-bit ALU & Superscalar ALU

📌 Overview

Designed and implemented a 4-bit ALU and Superscalar ALU using Verilog on FPGA.

⚙️ Features

- 16 operations (Arithmetic + Logic)
- Parallel execution using dual ALUs
- FPGA implementation (Spartan-6)

🛠️ Tools

- Verilog HDL
- Xilinx ISE
- FPGA (Atlys)
